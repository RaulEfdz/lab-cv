**Ubicación:** Bogotá, Colombia (Híbrido)
**Tipo:** Full-time

Buscamos PM/Scrum Master para liderar proyectos de transformación digital para clientes Fortune 500.

- 4+ años como Project Manager o Scrum Master

- Experiencia con Jira, Confluence, Azure DevOps
- Conocimiento técnico (no necesita programar, pero debe entender arquitectura)

- Comunicación con clientes
- Reportes de avance y métricas
- Coaching a equipos

## Keywords ATS
Project Management, Scrum Master, Agile, Scrum, Jira, Confluence, PMP, CSM, Stakeholder Management, Risk Management, Sprint Planning, Retrospective, IT Consulting
