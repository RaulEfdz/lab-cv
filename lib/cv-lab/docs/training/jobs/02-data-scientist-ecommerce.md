# Job: Data Scientist - E-commerce
**Empresa:** MegaStore Online
**Ubicación:** Ciudad de México (Híbrido)
**Tipo:** Full-time

## Descripción
Buscamos Data Scientist para optimizar nuestras recomendaciones de productos y predicción de demanda. Procesamos datos de 10M+ usuarios mensuales.

## Requisitos
- 3+ años en ciencia de datos o machine learning
- Python (pandas, numpy, scikit-learn)
- Experiencia con TensorFlow o PyTorch
- SQL avanzado
- Experiencia con big data (Spark, Hadoop)
- Conocimiento de A/B testing
- Visualización (Tableau, PowerBI)
- Experiencia en retail o e-commerce (deseable)

## Responsabilidades
- Desarrollar modelos de recomendación
- Análisis de comportamiento de usuarios
- Predicción de demanda y inventario
- Presentar insights a stakeholders

## Keywords ATS
Data Science, Machine Learning, Python, TensorFlow, PyTorch, pandas, SQL, Spark, A/B testing, Tableau, recommendation systems, e-commerce, retail, analytics

## Salario
$3,500 - $5,500 USD/mes
